How to export this as og-image.png:
1. Open this file in Chrome.
2. DevTools (Cmd+Opt+I) → Cmd+Shift+P → type "Capture node screenshot".
3. Click the .canvas element in Elements panel, then run the command.
4. Save the resulting PNG as public/og-image.png.
(Or just take a 1200×630 area screenshot of the canvas with Cmd+Shift+4 → Space.)
