# LaTeX Support in AP Prep Hub

The AP Prep Hub now supports LaTeX math rendering in flashcards! This allows you to create beautiful mathematical expressions and formulas.

## How to Use LaTeX

### Inline Math
For inline mathematical expressions, wrap your LaTeX code with single dollar signs:
```
The derivative of $x^2$ is $2x$
```

### Block Math
For display equations (centered and larger), use double dollar signs:
```
The quadratic formula is: $$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$
```

## Common LaTeX Examples

### Basic Operations
- Addition: `$a + b$` → $a + b$
- Subtraction: `$a - b$` → $a - b$
- Multiplication: `$a \times b$` or `$a \cdot b$`
- Division: `$\frac{a}{b}$` → $\frac{a}{b}$

### Exponents and Roots
- Exponents: `$x^2$`, `$x^{n+1}$`
- Roots: `$\sqrt{x}$`, `$\sqrt[n]{x}$`

### Greek Letters
- `$\alpha$`, `$\beta$`, `$\gamma$`, `$\delta$`
- `$\pi$`, `$\theta$`, `$\phi$`, `$\lambda$`

### Common Functions
- Trigonometric: `$\sin(x)$`, `$\cos(x)$`, `$\tan(x)$`
- Logarithms: `$\log(x)$`, `$\ln(x)$`
- Limits: `$\lim_{x \to \infty} f(x)$`

### Calculus
- Derivatives: `$\frac{dy}{dx}$`, `$f'(x)$`
- Integrals: `$\int f(x) dx$`, `$\int_a^b f(x) dx$`

### Chemistry
- Subscripts: `$H_2O$`, `$CO_2$`
- Arrows: `$A \rightarrow B$`
- Equilibrium: `$A \rightleftharpoons B$`

### Physics
- Vectors: `$\vec{F}$`, `$\hat{n}$`
- Units: `$v = 10 \text{ m/s}$`

## Examples in Flashcards

The AI will automatically generate LaTeX when creating flashcards for math and science subjects. You can also manually add LaTeX expressions when creating custom flashcards.

### Sample Questions with LaTeX:
1. **Question**: What is the quadratic formula?
   **Answer**: $$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

2. **Question**: What is the derivative of $\sin(x)$?
   **Answer**: $\frac{d}{dx}\sin(x) = \cos(x)$

3. **Question**: Express Newton's second law
   **Answer**: $$\vec{F} = m\vec{a}$$

## Tips
- Always preview your flashcards to ensure LaTeX renders correctly
- Use `\text{}` for text within math expressions
- For complex expressions, consider breaking them into multiple lines
- If LaTeX fails to render, the raw text will be shown in red
